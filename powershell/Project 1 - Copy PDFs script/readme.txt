Purpose of Script:

Scans all folders within user profile, excluding OneDrive files, for PDFs and creates copies in a new folder.

Instructions:

-Run the script
-It will ask you to name the new folder that will be created to house your PDFs
-It will print a list of PDFs that it scanned
-Then, it will ask you if you want to copy these files to a new folder. Select Yes to proceed. Select No to terminate the script.

-If you select Yes, it will copy the files to the newly created folder, and then it will ask if you want to delete the old files. 

BEWARE: Selecting Yes will render the old files unrecoverable. Verify that files have copied before proceeding.

If you select No, the script will terminate.
